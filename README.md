# SorTech-SistemaDeGestionDeProductos
SorTech es un sistema de gestión de productos en WPF con C#, diseñado para tiendas de electrónica. Implementa estructuras de datos avanzadas, algoritmos eficientes y una interfaz alineada con principios de usabilidad. Fue desarrollado en la Práctica 4 del curso de Técnicas de Programación, aplicando fundamentos de computación en un entorno real.
